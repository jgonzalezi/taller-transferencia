# Software Taller de Transferencia Tecnológica INDUMIL


Enlaces de descarga al software a usar en la sesión práctica.

Recordar que se necesita un computador con sistema operativo Windows 11

0. MATLAB + MATLAB UAV Toolbox - Software para procesar los datos
   Por favor instalar UAV Toolbox en su entorno MATLAB.

1. [RFDTools](https://rfdx.atlassian.net/wiki/download/attachments/242712577/RFDTools_4.0.18.0_x64.msix?api=v2) - Software para configurar radios RFD900x
2. [RFD SiK V3.57 rfd900xv2](https://files.rfdesign.com.au/Files/firmware/RFDSiK%20V3.57%20rfd900x2.gbl) - Firmware para el radio.
3. [QGroundControl](https://d176tv9ibo4jno.cloudfront.net/latest/QGroundControl-installer.exe) - Software de estación en tierra y configuración del CubeOrange.
4. [MissionPlanner](https://firmware.ardupilot.org/Tools/MissionPlanner/MissionPlanner-latest.msi) - Software de estación en tierra.
